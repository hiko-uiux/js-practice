feat：新機能
refactor: リファクタリング

fix：バグ修正
perf：パフォーマンスを向上させるコード変更
///
test: テスト修正
build：ビルドシステムまたは外部の依存関係に影響を与える変更
ci：CI構成ファイルとスクリプトの変更
doc：ドキュメントのみが変更されます
